Objective

Learn how to load, inspect, and visualize a dataset to understand trends, distributions, and outliers.

Dataset

Iris Dataset (CSV format)

Can be loaded directly using Seaborn or downloaded as CSV.

Instructions

Load the dataset using pandas.

Inspect the data:

Print shape, column names, and first few rows using .head().

Use .info() and .describe() for summary statistics.

Visualize the dataset:

Scatter plots to show relationships between features.

Histograms for value distributions.

Box plots to detect outliers.

Use Matplotlib and Seaborn for plotting.

Skills Gained

Loading and inspecting datasets with pandas

Computing descriptive statistics

Basic plotting and data visualization
